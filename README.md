# SECURNECT
Command-line interface encrypted chat manager.

# Overview

Securnect is a command-line interface (CLI) application that empowers users to create and utilize secure text chats with double asymmetric encryption. This tool ensures the confidentiality and integrity of your communication by employing robust encryption techniques. Additionally, Securnect supports multiple users, allowing each individual to manage their own set of encrypted chats.

# Features

- Double Asymmetric Encryption:
     Securnect employs a sophisticated double asymmetric encryption mechanism to ensure the security and privacy of your text chats.

- Multiple User Support:
     Users can create and manage their own set of encrypted chats, providing a personalized and secure communication experience.

- Command-Line Interface:
     The CLI provides a simple and efficient way to interact with Securnect, making it accessible for both beginners and advanced users.

# Installation

To install Securnect, follow these steps:

- Clone the Securnect repository:
     git clone https://github.com/Just-A-Regular-Guy/securnect

- Navigate to the Securnect directory:
     cd securnect

- run Securnect:
     python3 securnect.py
